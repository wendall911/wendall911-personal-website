As a cloud security company, [Cenzic](https://www.trustwave.com/en-us/company/newsroom/news/trustwave-acquires-cenzic-to-enhance-cloud-based-security-testing-platform/), acquired by Trustwave in 2014, needed a trusted partner to deploy software keys to its commercial clients. Deploying software keys to secure server environments required sophisticated integration into an existing web infrastructure and careful planning to ensure the security and reliability of the system. This system needed to be 100% reliable, as it would reflect poorly on a security company to have a flaky or insecure product registration system.
